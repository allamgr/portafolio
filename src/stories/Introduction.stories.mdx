# Introduction to portafolio

**portafolio** is a react library that can be used to build your Resume/CV/Portfolio very fast with only a <code>JSON</code> file with the required structure by the library or just passing an <code>Object</code> in the component <code>props</code>.

**portafolio** helps you build your **personal portfolio website** using default templates or using the components individually depending on your needs.
As developers we don't like to spend too much time creating or updating our personal porftolio, this was my case and I decide to make a reusable and extensible library in React with the purpose of use it in my own porfolio website, and why not for all developers/designers out there.
