# Templates

If you want to save more time, **portafolio** has default **templates** predesigned with all possible components, and the information is displayed dynamically.

## How to use

```js
# App.js
import { TemplateLean } from '@allamgr/portafolio'
import jsonData from './cv.json';
```

## Design custom templates

You can design your own templates and include only the components you need on your portfolio.

```js
import React from "react"
import {
  PersonalInfo,
  WorkExperience,
  SkillsAndTech,
  EducationInfo,
} from "@allamgr/portafolio"

export const MyCustomTemplate = (props) => {
  return (
    <div className="custom-template">
      <PersonalInfo {...props.personalInfo} />
      <SkillsAndTech {...props.skillsAndTech} />
      {props.workExperience && <WorkExperience {...props.workExperience} />}
      {props.educationInfo && <EducationInfo {...props.educationInfo} />}
    </div>
  )
}

export default MyCustomTemplate
```

**Note:** you can collaborate and create a pull request with your template, I will evaluate if can be added to the library as a default template.
