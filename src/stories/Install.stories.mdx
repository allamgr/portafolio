# Install portafolio

portafolio v0.3 requires **React 17.0.2 or later.**

To use portafolio with your React app, install it as a dependency:

```sh
# If you use npm:
npm install --save @allamgr/portafolio

# Or if you use Yarn:
yarn add @allamgr/portafolio
```

## How to use

# App.js
